# Hospital Emergency Room Dashboard (Interactive creation using power bi)

## Overall Project Objective 
 The objective of this project is to develop a Hospital Emergency Room Analysis Dashboard in Power BI that provides clear insights into emergency room operations. By analyzing patient flow, wait times, admissions, referrals, and other key metrics, the dashboard helps stakeholders monitor performance, improve operational efficiency, and make data-driven decisions to enhance patient care and resource management.


## Dataset Used 
 <a href="https://github.com/jeevanantham0407/Data-Analysis-Dashboard/blob/61b588011d05a4d18b73f2128f5ee9fa2bbed38a/Hospital%20ER_Data.csv">Dataset</a>

## KPI’s Requirements
•	 Number of Patients
•	 Average Wait Time
•	 Patient Satisfaction Score
•	 Number of Patients Referred

## Dashboard 1: Monthly View 
Objective: Monitor key metrics and trends on a month-by-month basis to identify patterns and areas for improvement.
Charts to Develop:
•	Patient Admission Status: Track admitted vs. non-admitted patients.
•	Patient Age Distribution: Group patients by 10-year age intervals.
•	Department Referrals: Analyze referral trends across different departments.
•	Timeliness: Measure the percentage of patients seen within 30 minutes.
•	Gender Analysis: Visualize patient distribution by gender.
•	Racial Demographics: Analyze patient data by race.
•	Time Analysis: Assess patient volume by day and hour.


## Dashboard 2: Consolidated View 
Objective: Provide a holistic summary of hospital performance for a selected date range.
Charts to Develop: Similar metrics as the Monthly View, but aggregated over a customizable date range for broader insights and trend analysis

## Dashboard 3: Patient Details
Objective: Offer granular insights into patient-level data to enable detailed analysis and troubleshooting.
Charts to Develop: A grid displaying essential fields:
•	Patient ID
•	Patient Full Name
•	Gender
•	Age
•	Admission Date
•	Patient Race
•	Wait Time
•	Department Referral
•	Admission Status


## Process 
Data Collection – Imported the Hospital Emergency Room dataset into Power BI.
Data Cleaning – Cleaned and transformed the data using Power Query by handling missing values, correcting data types, and removing inconsistencies.
 Data Modeling – Established relationships between tables and created a star schema for efficient analysis.
DAX Calculations – Developed measures and calculated columns to analyze KPIs such as total patients, average wait time, admission rate, and patient satisfaction score.
Dashboard Development – Designed interactive dashboards with cards, charts, slicers, and matrices to visualize emergency room performance.
Performance Analysis – Analyzed patient flow, referrals, wait times, demographics, and departmental trends to generate actionable insights.

# Project Insights

## Dashboard 1: Monthly View
This dashboard provides month-wise insights into emergency room performance, enabling users to monitor operational efficiency and patient trends.
Insights
•	Tracks monthly patient volume and admission status. 
•	Measures average wait time and patients seen within the 30-minute target. 
•	Evaluates patient satisfaction for the selected month. 
•	Identifies patient distribution by age group, gender, and race. 
•	Highlights department referral patterns. 
•	Reveals peak patient visits by day and hour to support resource planning.




 


## Dashboard 2: Consolidated View
This dashboard presents a comprehensive overview of emergency room performance across a selected date range, supporting strategic analysis and decision-making.
Insights
•	Summarizes key performance indicators, including total patients, wait time, satisfaction score, and referrals. 
•	Compares admission and non-admission trends over time. 
•	Analyzes demographic distribution and referral patterns. 
•	Monitors compliance with the 30-minute service target. 
•	Identifies long-term trends in patient volume and operational performance. 
•	Supports staffing and resource optimization by analyzing peak patient traffic. 

## Dashboard 3: Patient Details
This dashboard provides detailed patient-level information for operational reporting and in-depth analysis.
Insights
•	Displays complete patient records, including demographics and admission details. 
•	Enables filtering and exploration of individual patient information. 
•	Helps identify patients with longer wait times. 
•	Tracks department referrals and admission status at the patient level. 
•	Supports data validation, reporting, and detailed operational analysis.
